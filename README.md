# 🔐 Password Generator

## 📌 Project Description

The Password Generator is a Python desktop application built using **Tkinter**. It generates strong and secure passwords based on the user's selected options. The application also checks password strength and allows users to copy the generated password.

This project was developed as part of the **Oasis Infobyte Internship (OIBSIP)**.

---

## ✨ Features

- Generate secure passwords
- Select password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Copy password to clipboard
- Password strength indicator
- Password history
- Light and Dark mode
- Simple and easy-to-use interface

---

## 🛠️ Technologies Used

- Python
- Tkinter
- Secrets Module
- String Module
- Pyperclip

---

## ▶️ How to Run

1. Download or clone the project.
2. Open the project folder.
3. Install the required package:

```bash
pip install pyperclip
```

4. Run the following command:

```bash
python password_generator_gui.py
```
